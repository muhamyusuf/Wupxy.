## Introduction

TweetLineer is a website for informatics engineering students to share their ideas, opinions, criticism about anything in informatics engineering area. This website was build from scratch with fullstack logic.

## Tech Stack

- [Next.js](https://nextjs.org) as React Framework and TypeScript as language
- [Tailwind CSS](https://tailwindcss.com) and [shadcn/ui](https://ui.shadcn.com) for styling
- [Prisma](https://prisma.io) as ORM
- [NextAuth.js](https://next-auth.js.org) for authentication
- [PlanetScale](https://planetscale.com) as database
- [Vercel](https://vercel.com) for deployment
