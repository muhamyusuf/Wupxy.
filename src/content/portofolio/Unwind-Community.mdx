## Introduction

Unwind community is a reddit clone but with modification, that allows users to create post and join a community. The application is built using . The application is deployed on Vercel.

## Features

- Create a post
- Join a community
- Upvote and downvote a post
- Comment on a post
- View a post, community and user profile.
- Settings page to update username
- Change theme(dark and light)

## Tech Stack

- Next.js
- Tailwind CSS
- Planetscale
- Vercel
- Uploadthing
- Redis
